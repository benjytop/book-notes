# 决策

## 命题

> **作者主张**：决策无处不在——生活就是所有"选择"的总和（来源：阿尔贝·加缪，引自前言）。

本书把"决策"定义为：在多个可选方案中**选择** + **做决定**。这是一项**可分解、可训练、可学习**的能力——不是天赋独占。

## 决策的 4 类角色

| 角色 | 核心 | 例子 |
|---|---|---|
| 棋手（决策角色之一） | 形势评估 — 从何着手 | 行业重组面对新挑战，公司应该先做什么？ |
| 侦探 | 原因分析 — 找出根本原因 | 为什么员工对变革抵触？ |
| 勇士 | 果断出手 — 资源有限时最优动作 | 火灾应急决策 |
| 法官 | 裁决 — 信息综合后给出最终判定 | 多方证据衡量后判决 |

## 决策的过程元素

1. **决策者**（角色）
2. **决策场景**（目的 / 时机）
3. **决策核心**（核心命题）
4. **决策行动**（具体动作）

## 证据 / 案例

- 大众尾气案：每个角色都在场，但"勇士 + 法官"压过"侦探 + 棋手"——结果失衡。
- 诺曼底：气象学家作为"侦探"被压制——沉默文化让决策失去纠偏。
- 雅典瘟疫中的苏格拉底："勇气 ≠ 鲁莽"——法官 + 棋手双轮驱动。

## 分析

本书把"决策"还原为**四角色 + 四步流程**——一个工程化的拆解。这与卡尼曼《思考，快与慢》的双系统分析兼容，但本书把"流程中心"放到具体管理场景。

## 限制与反例

- **决策在压力下可能变得非理性**——本书承认这是框架的局限
- **真正无信息的两难困境**——4 类角色未必能覆盖（道德选择）
