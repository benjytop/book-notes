# 非自由主义

## 定义

> Illiberal — 拒绝平等 / 多元文化 / 法律程序的实践集合。包括: 奴隶制 / 契约奴 / 排华 / 工会镇压 / 联邦政府集权 / 多数暴政 / etc.

## 核心命题

> 作者主张'非自由主义在美国从来不是异端, 而是贯穿乃至塑造美国历史的一条幽暗脉络'。

## 证据与案例

本书核心命题: 从殖民地到 BLM 时代这条线都贯穿。

## 限制与反例

非自由主义 = 标签, 但包含太多 — 缺乏具体内容时易被模糊化。

全书 + 导论
