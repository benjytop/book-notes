# 凝视的四种类型

> 第六部第 23 节的核心概念。

## 四类

| 类型 | 描述 | 例子 |
|---|---|---|
| 第一类 | 寻求"公众目光" | 德国歌手、美国女明星、被捕记者 |
| 第二类 | 寻求"熟人目光" | [[玛丽-克洛德]]、[[玛丽-安娜]] |
| 第三类 | 寻求"所爱之人的目光" | [[托马斯]]、[[特蕾莎]] |
| 第四类 | 寻求"想象中不在身边的人的目光" | [[弗兰茨]]、[[西蒙]] |

## 推论

- 第一类与第三类同样危险。
- 第二类最快活，因为他们最终能获得某种目光。
- 第四类最少见，是梦想家。

## 涉及人物

- 德国歌手、美国女明星、记者（[[记者（驼背）]]）：第一类
- [[玛丽-克洛德]]、[[玛丽-安娜]]：第二类
- [[托马斯]]、[[特蕾莎]]：第三类
- [[弗兰茨]]、[[西蒙]]：第四类
