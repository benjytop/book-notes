# 人口密度经济

## 命题

> **作者主张**：人口密度越高 → 分工越细 → 经济效率越高 — 这是大城市集聚的根本原因。

## 核心论点

集聚 → 降低交易成本 → 提高劳动生产率。

## 证据

东京 / 纽约 / 珠三角 / 长三角等高密度地区经济数据。

## 限制与反例

高密度也带来拥堵、污染等外部成本 — 需要城市规划配套。
