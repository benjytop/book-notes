# 全书结构

## 一句话核心命题

**情绪自控 = 6 激素按比例调配 + 50+ 工具的日常化应用 + 自我领导力心态**。

## 9 章 + 序言 主题词

| 部分 | 主题词 |
|---|---|
| 引言 | 一个阴沉秋日的转机 |
| 第 01 章 | 多巴胺 — 驱动力和愉悦 |
| 第 02 章 | 催产素 — 连接与信任 |
| 第 03 章 | 血清素 — 满足感与好心情 |

| 第 08 章 | 痛苦的日子为何难以扭转 |
| 第 09 章 | 迎接新生活 |

## 核心论点（10 条）

1. **多巴胺是驱动力 + 愉悦的源泉** — 但过量会上瘾
2. **催产素塑造关系** — 是相互的正反馈
3. **血清素带来满足** — 食物 + 阳光 + 运动是基础补给
4. **皮质醇短时是朋友，长期是敌人** — 节律控制是关键
5. **内啡肽把痛苦转化为愉悦** — 是 runner high 的来源
6. **睾酮不只男性专属** — 是任何人都可"激活的赢家激素"
7. **6 激素按比例调配 = 鸡尾酒** — 比例因人因时
8. **自我领导力 ≠ 自律** — 是主动配方设计师
9. **负面循环是自我增强** — 一旦形成极难扭转，必须主动打破
10. **50+ 工具 = 50+ 鸡尾酒配料** — 用日常持续性使用
