## 一句话定位

> 见 [[读书脑图]]。

## 这本书在处理什么问题

全球经济秩序正在经历"百年未有之大变局"。本书尝试回答：

1. **全球化波折是方向性的，还是周期性的？**——市场本位 vs 国别本位
2. **企业当下的"寻路难题"是什么？**——作业不多了、市场卷了、底盘不稳
3. **行路难如何解锁？**——平常功夫 + 宽深快严

## 阅读前要知道的背景

- **作者**：周其仁（北大经济学教授 / 国家发展研究院院长）
- **学术谱系**：芝加哥 + 杜润生一脉，强调制度成本 / 产权 / 全球化
- **写作时间**：本书文章跨度 2019–2025，包含多次论坛 / 财新周刊访谈
- **本书主线**：国别本位（关税 / 政治 / 民族）与市场本位（自由贸易 / 跨境企业 / 要素流动）的张力

- **第一次**：检视阅读 + 重点精读（[[阅读方式]]）

## 带着这五个问题阅读

1. 你怎么理解"国别本位"vs"市场本位"的张力？
2. **贸易战的核心问题**是什么？真的是输赢问题吗？
3. "平常功夫"中的"宽 / 深 / 快 / 严"分别意味着什么？
4. 中国制造业的下一个增长点在哪里？
5. 改革为何是"最重要的投资"？

## 可能的阅读难点

- **第 5 部分是纪念性文章**，与"如何思考"无关——是"如何回忆"
- **经济史话题**对非专业人士门槛较高，但周其仁的文风偏口语化，可通读
- **多地名 + 时间跨度** 容易让读者迷失——看 [[章节地图]] 引导

## 建议记录方式

- 用 [[复习卡片]] 抽查 5 大命题
- 把"国别 vs 市场"作为分析任意国际事件的透镜
- 用 [[主题与核心观点]] 提炼作者的核心论点
