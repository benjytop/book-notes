# 伊甸园与循环时间

> 第七部第 4 节的核心概念。

## 命题

> **作者主张**："伊甸园的生活不同于那将我们引向未知的直线赛跑，也不是一次历险。它是在已知的事物中间循环移动。它的单调并非厌烦，而是幸福。"

- 人类时间 = 直线
- 动物时间 = 循环
- 伊甸园 = 循环 + 单调 = 幸福

## 关键论证

- 亚当对着泉水俯下身，还不知道水中看到的就是自己。
- 亚当就像卡列宁：卡列宁看镜子不认得自己的影子。
- 人被逐出伊甸园 → 时间变成直线 → 不可能幸福。
- 狗从未被逐出 → 循环时间 → 接近幸福。
