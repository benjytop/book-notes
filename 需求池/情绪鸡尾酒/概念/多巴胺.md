# 多巴胺

## 定义

> 一种奖赏 + 驱动力神经递质。在进食 / 性 / 完成小目标时释放，给我们追求感和愉悦感。

## 核心命题

> 你需要新的刺激让多巴胺反复上升 — 不能靠单一刺激（暴食 / 短视频上瘾）。

## 工具 / 证据

8+ 工具：晨跑、听喜欢的音乐、完成小任务、新鲜事、关键对话、暂时奖励、规律运动、新颖活动。

## 限制与反例

过多 = 上瘾 + 注意力飘忽；过少 = 没动力 + 抑郁；管控 = 节奏性满足。
