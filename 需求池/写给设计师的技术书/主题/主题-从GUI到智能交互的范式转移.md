# 主题:从 GUI 到智能交互的范式转移

## 命题

> **作者主张**:设计师需要从「基于屏幕的图形用户界面」思维,转变为「空间交互及智能交互」的设计思维。这是一次「终端形态 + 交互方式 + 内容生产」的三重迁移。

## 主要论据

### 1. 终端形态迁移

| 时代 | 主导终端 | 设计师关注点 |
|---|---|---|
| 桌面时代 | 鼠标 + 键盘 + 显示器 | 像素级布局、菜单层级 |
| 移动时代 | 触摸屏手机 | 单手操作、滚动设计、App 信息架构 |
| 多终端时代(2020+) | 折叠屏 / Pad / 桌面 / 车机 / XR | 动态布局、多窗口、跨设备一致性 |
| 智能时代(2023+) | AI 助手 / 智能音箱 / XR | 意图理解、自然对话、传感器融合 |

### 2. 交互方式迁移

```text
GUI 时代:
  触摸 → 滑动 → 点击 → 输入文字
                              ↓
智能交互时代:
  触摸 + 语音 + 手势 + 人脸 + 眼动 + 姿态
       ↑ 多模态组合,意图驱动
```

### 3. 内容生产迁移

- **GUI 时代**:内容由人工生产,设计师排版
- **智能时代**:AIGC 自动生成内容,设计师编辑与策展
- 作者在前言直接说「2023 年对于设计师来说是开启新篇章的一年,无论是 AIGC、Web 3 还是 XR 都将带来新的机会点」

## 关键转折点

- **2020 年**:中国造车新势力元年,大量互联网设计师进入汽车行业
- **2022 年 11 月**:工信部等五部门发布《虚拟现实与行业应用融合发展行动计划(2022—2026 年)》,XR 政策红利
- **2023 年(本书出版年)**:AIGC 浪潮起点,设计师工作流被重塑

## 设计含义

- **不再只是画界面**:设计师输出物还包括传感器布局、算法效果评估、跨端一致性策略
- **从「流程设计」到「意图设计」**:用户表达意图,系统理解并执行
- **从「确定性」到「不确定性」**:AI 输出有不同概率,设计要处理置信度
- **从「像素」到「空间」**:XR 时代设计对象是 3D 空间,不是 2D 像素
