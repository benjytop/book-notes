# 韦启昌

**角色 / 身份**：台湾 / 香港翻译家 / 学者

## 简介

上海人民出版社的人生智慧、叔本华系列的长期译者和校注者。

## 在本书中的贡献

翻译了叔本华的一系列著作 — 包括《人生的智慧》《叔本华美学随笔》《叔本华论说文集》等。
