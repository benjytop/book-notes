# 引子：内容提要 / 推荐语 / 前言

## 一句话定位

> "决策无处不在。" — 阿尔贝·加缪（本书前言引用）

## 这一部分的用途

- **内容提要**：出版社用 4 个要点概括本书卖点
- **推荐语**：从业者推荐 + 同行呼应当
- **前言**：作者自己谈"为何要读这本书"

## 关键概念钩子

| 钩子 | 出处 | 含义 |
|---|---|---|
| 决策的质量 = 管理者的关键指标 | 推荐语 | 高质量决策成为可衡量指标 |
| 决策 = 认知神经科学 + 企业文化 | 推荐语 | 个人能力 + 组织能力双侧发力 |
| 大众尾气门 | 推荐语 + 前言 | 过度自信 + 拒听反对的典型案例 |
| 加缪 / 孙子引言 | 前言 | "选择是生活全部" + "庙算多算胜" |
