# 全书结构

## 一句话核心命题

**高质量决策的关键在"流程"而非"结果"**——任何决策都是"流程中心"的，决策者通过"一个中心 + 4 步流程"才能持续纠偏、摆脱认知盲区。

## 五大部主题词

| 部 | 主题词 |
|---|---|
| 01 | 决策的核心 |
| 02 | 决策的天敌 |
| 03 | 决策的生理机制 |
| 04 | 如何做出高质量决策 |
| 05 | 决策的策略和核心技巧 |

## 核心论点

1. **决策是一项可分解、可训练、可学习的能力**——不是天赋、也不是经验独占。
2. **错误的本质是 偏见 + 沉默文化 + 过度自信**——三者叠加产生"决策陷阱"。
3. **纠偏要靠流程**：一个中心（流程为决策中心）+ 4 步（发 / 检 / 展 / 收）+ 广开言路。
4. **生理机制是基底**：大脑的双系统（斗士 vs 侦察兵 / 系统 1 / 系统 2）决定了决策风格。
5. **组织级的纠偏比个人级更重要**——决策从来不是"一个人"的事。
