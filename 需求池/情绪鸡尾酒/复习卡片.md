**2** Q: 本书的 6 激素？
A: 多巴胺 / 催产素 / 血清素 / 皮质醇 / 内啡肽 / 睾酮。

**3** Q: 本书中文版字数（去空白）？
A: 全文字数 128,779 字符。

**4** Q: 本书的副标题？
A: 从多巴胺到内啡肽，调配幸福秘方。

## 概念（6 激素）

**6** Q: 多巴胺的主要作用？
A: 驱动力 + 愉悦 + 奖励机制。

**7** Q: 多巴胺过量的问题？
A: 上瘾 + 注意力飘忽 + 难以欣赏最初的活动。

**8** Q: 催产素的主要作用？
A: 连接 + 信任 + 同理心。

**9** Q: 催产素为什么是自我增强的？
A: 你显得值得信赖 → 别人也更愿意信任你。

**10** Q: 血清素的核心功能？
A: 满足感 + 好心情 + 节律稳定性。

**11** Q: 血清素的基础补给？
A: 阳光 + 5-HTP 食物（香蕉 / 坚果）+ 慢节奏消化。

**12** Q: 皮质醇的节律？
A: 晨高 → 上午峰值 → 下午下降 → 凌晨最低。

**13** Q: 皮质醇短期 vs 长期的影响？
A: 短期是朋友（应急），长期是敌人（焦虑 / 失眠 / 肥胖）。

**14** Q: 内啡肽的来源？
A: 剧烈运动 / 辣食 / 大笑 / 性 — Runner high。

**15** Q: 睾酮的"性别中立"立场？
A: 睾酮是自信 / 果断 / 竞争激素 — 不限性别。

## 核心命题

**16** Q: 自我领导力是什么？
A: 决定吃什么 / 何时睡 / 与谁交 / 怎么想 — 不是被社会框架约束。

**17** Q: 鸡尾酒模型的核心？
A: 6 种激素按比例调配 = 你的鸡尾酒 — 改变输入改变配方。

**18** Q: 魔鬼鸡尾酒？
A: 压力 → 皮质醇长期高 → 睡眠差 → 暴食 → 多巴胺不稳 → 压力。

**19** Q: 4-7-8 呼吸是怎么做？
A: 吸气 4 秒 + 屏 7 秒 + 呼 8 秒 → 副交感神经激活。

**20** Q: 50 工具分布在哪些维度？
A: 食物 / 运动 / 关系 / 思维 / 呼吸 / 阳光 / 节律 / 习惯 / 庆祝。

## 应用

**21** Q: 晨跑为何提升多巴胺？
A: 完成感 + 行为方式 + 阳光 → 三重多巴胺补给。

**22** Q: 拥抱如何提高催产素？
A: 身体接触触发生理反射 + 心率同步 + 信任感。

**23** Q: 4-7-8 呼吸对皮质醇有什么影响？
A: 激活副交感神经 → 缓和"战斗或逃跑"反应 → 皮质醇下降。

**24** Q: 感恩日记如何提高血清素？
A: 把注意力从问题 → 好处 → 多巴胺 + 血清素双向补给。

**25** Q: 胜利姿态为何提高睾酮？
A: 物理姿态 → 大脑信号 → 激素释放 → 自信反哺。

## 局限

**26** Q: 本书的科学边界？
A: 工具化倾向 — 不替代医学干预（重度抑郁 / PTSD / 双相）。
