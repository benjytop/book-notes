# 全书结构

## 一句话核心命题

**RPG 化的 7 天职业探索 — 强项 + 价值观 + 理想未来 = 7 天做完美的一份'理想职业地图'。**

## 7 天路径

| 天 | 主题 | 工具 |
|---|---|---|
| 第 1 天 | 决定游戏的结局（理想未来） | 价值观排序 |
| 第 2 天 | 选择关卡 | 任务清单 |
| 第 3 天 | 发现先天强项 | 15 个发现技巧 |
| 第 4 天 | 发现后天强项 | 持续练习的技能 |
| 第 5 天 | 强化强项 | 练习 + 反馈 |
| 第 6 天 | 制作地图 | 终点 + 节点 + 主线任务 |
| 第 7 天 | 完善地图 | 敌人 + 强项 + 隐藏任务 |

## 核心论点（8 条）

1. 每个人都有'先天 + 后天'两类强项
2. 大部分人没意识到自己的强项
3. 强项需要主动塑造 + 主动使用
4. 理想未来 = 价值观 + 形象 + 物质
5. 价值观 = 多重的可排序 — 不是单一最大化
6. 强项 × 市场需求 = 双轴交集
7. 玩家体验谈 = 真实案件
8. 灰心时 5 条原则 = 工具书的额外救济
