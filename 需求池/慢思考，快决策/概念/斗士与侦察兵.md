# 斗士与侦察兵

## 命题

> **作者主张**：决策中并存两种思维模式——**斗士（fighter）** 和 **侦察兵（scout）**。两者必须同时使用，缺一不可。

这是卡尼曼"系统 1 / 系统 2"的工程化比喻。

## 两模式的对比

| 维度 | 斗士 | 侦察兵 |
|---|---|---|
| 角色 | 战斗中的士兵 | 走出营地收集情报 |
| 任务 | 攻击 / 防守 | 识别风险 |
| 思维 | 反射 / 直觉 / 反应 | 诚实评估 / 好奇 / 审视 |
| 时机 | 紧急决策 | 长期决策 |
| 生理基础 | 杏仁核主导 | 前额叶主导 |

## 证据 / 案例

> 一支精良的队伍中，斗士和侦察兵都是必不可少的。但你也可以将它们想象为两种不同的决策思维模式——一种关于在日常生活和工作中如何处理信息并做出决定的两种不同风格的类比。

> 在日常决策的过程中，你更像是一名斗士还是侦察兵呢？

## 分析

本书的核心洞见之一：**决策质量取决于两个模式的比例**，而不是只用某一种。

- 高质量决策 = 斗士的"快速反应" + 侦察兵的"诚实评估"

## 限制与反例

- 在**高压力**时间紧的场景下，侦察兵的"审慎"会变成负担
- 在**高模糊**信息稀缺的场景下，斗士的"反射"缺乏刺激源
