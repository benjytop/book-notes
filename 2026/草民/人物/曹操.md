# 曹操

## 简介

**曹操** 是本书首篇同名短篇《曹操背观音去了》的主角. 东石镇驼背老人. 每天从西码头走到东码头, 背着两个背篓 (海鲜 + 观音像). " 成佛" 后全镇讨论给他建庙. 蔡崇达笔下" 草民" 的代表.

## 关键事件

1. **每天往返东西码头** - 讨小海 + 卖香
2. **用肥皂洗手** - 仪式感
3. **背观音像** - 信仰
4. **成佛了** - 镇上传说
