# 贸易战

## 命题

> **作者主张**：贸易战起于国别利益分歧，一旦开打，多半循"你一拍、我一拍"逻辑；双方企业 / 消费者都受害，没有赢家。

## 核心论点

理论最优（单边自由贸易）在政治上不可持续，因此贸易战难断。

## 证据

2018-2019 中美关税战 → 双方进口商 / 消费者双受害；ASML / NVIDIA 限制出口 → 自身收入受损。

## 限制与反例

短期内贸易战可能给本土企业带来"机会窗口"（如苹果将产能转印度）。
