# 14 类图书分类法 (Book Type Taxonomy)

## 概述

为图书分析提供 14 大类分类法。每一类决定 vault 结构、抽取任务、Mermaid 模板。

| # | 大类 (T 类型) | 小类 (举例) | vault 子目录 | Mermaid 文件名 |
|---|---|---|---|---|
| **T1** | 文学 | 小说 / 短篇 / 散文 / 诗歌 / 纪实文学 | `人物/` `事件/` `地点/` | `人物关系图.mermaid.md` |
| **T2** | 思想史/哲学 | 哲学 / 思想史 / 思想评论 | `思想家/` `概念/` `思想流派/` | `思想影响图.mermaid.md` |
| **T3** | 方法论/自助 | 时间管理 / 学习方法 / 习惯养成 | `概念/` `方法/` `步骤/` | `概念图谱.mermaid.md` |
| **T4** | 投资/经济 | 价值投资 / 宏观 / 周期 / 经济学 | `思想家/` `理论/` `模型/` `案例/` | `理论关系图.mermaid.md` |
| **T5** | 商业/管理 | 商业 / 创业 / 管理 / 营销 | `原则/` `案例/` `框架/` | `框架图.mermaid.md` |
| **T6** | 政经/时政 | 政治评论 / 国际关系 / 政策分析 | `人物/` `事件/` `阵营/` `概念/` | `阵营-人物图.mermaid.md` |
| **T7** | 历史/纪实 | 历史 / 传记 / 游记 / 纪实 | `人物/` `事件/` `地点/` `概念/` | `时间线-人物图.mermaid.md` |
| **T8** | 心理/自助 | 心理学 / 情绪管理 / 自助 | `概念/` `理论/` `方法/` `案例/` | `概念图谱.mermaid.md` |
| **T9** | 科普/科学 | 科学 / 自然 / 技术 / 医学 | `概念/` `理论/` `案例/` `原理/` | `概念关系图.mermaid.md` |
| **T10** | 教育/学习 | 教育 / 育儿 / 学习方法 | `概念/` `理论/` `方法/` `案例/` | `概念图谱.mermaid.md` |
| **T11** | 艺术/设计 | 艺术 / 设计 / 建筑 / 摄影 | `人物/` `作品/` `风格/` `概念/` | `人物-风格图.mermaid.md` |
| **T12** | 育儿/家庭 | 育儿 / 家庭 / 健康 / 养生 | `概念/` `方法/` `案例/` | `概念图谱.mermaid.md` |
| **T13** | 旅行/地理 | 旅行文学 / 游记 / 地理 / 文化 | `地点/` `人物/` `事件/` `概念/` | `人物-地点图.mermaid.md` |
| **T14** | 混合 (跨大类) | 文学+哲学 / 历史+旅行 | 跨类型组合 | 类型化 (按实际混合) |

## 分类决策算法

```
Q1: 是叙事性 (有主角/情节) 还是非叙事性?
├── 叙事性 → Q2
│   ├── Q2: 主要内容是 1) 纯文学 2) 文学+哲学 3) 文学+历史 4) 文学+散文?
│   │   ├── 纯文学 → 文学 (T1)
│   │   ├── 文学+哲学 → 混合-文学+哲学 (T14)
│   │   ├── 文学+历史 → 混合-文学+历史 (T14)
│   │   └── 文学+散文 → 混合-文学+散文 (T14)
└── 非叙事性 → Q3
    ├── Q3: 主要实体是 1) 思想家 2) 概念 3) 投资 4) 商业原则 5) 政治人物 6) 历史事件 7) 心理 8) 科学 9) 教育 10) 艺术 11) 育儿 12) 旅行?
    │   ├── 思想家 → 思想史/哲学 (T2)
    │   ├── 概念+方法 → 方法论/自助 (T3)
    │   ├── 投资 → 投资/经济 (T4)
    │   ├── 商业原则 → 商业/管理 (T5)
    │   ├── 政治人物 → 政经/时政 (T6)
    │   ├── 历史/传记 → 历史/纪实 (T7)
    │   ├── 心理学 → 心理/自助 (T8)
    │   ├── 科学/自然 → 科普/科学 (T9)
    │   ├── 教育/育儿 → 教育/学习 (T10)
    │   ├── 艺术/设计 → 艺术/设计 (T11)
    │   ├── 家庭/健康 → 育儿/家庭 (T12)
    │   └── 旅行 → 旅行/地理 (T13)
```

## 关键判定规则

### T1 文学 (长篇小说)
- 有完整的主角与情节
- 关键词：小说、主角、故事线、章节叙事
- 例: 《灿烂千阳》《追风筝的人》《局外人》

### T14 混合 (含文学元素的非纯文学)
- 有叙事元素但主体是另一种内容
- 例:
  - 《局外人》: 文学 + 哲学 (存在主义)
  - 《带一本书去巴黎》: 文学 + 历史 (法国大革命)
  - 《我人生最开始的好朋友》: 文学 + 散文 (回忆性散文)
  - 《血与蜜之地》: 文学 + 历史/旅行 (巴尔干 30 年)
  - 《人生的智慧》/叔本华: 文学 (哲学随笔) → 归 T2 思想史/哲学

### T4 投资 vs T5 商业
- T4 投资: 个人投资/理财/市场分析/经济周期
- T5 商业: 创业/管理/营销/领导力
- 例: 《纳瓦尔宝典》= T5 (商业思维), 《周期》= T4 (投资)

### T7 历史 vs T13 旅行
- T7 历史: 严肃历史/传记/纪实 (含时间线)
- T13 旅行: 旅行文学/游记/地理
- 《带一本书去巴黎》= T14 (文学+历史)，但旅行文学如《旅行的艺术》= T13
