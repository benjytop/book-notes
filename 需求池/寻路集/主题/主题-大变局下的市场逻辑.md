# 大变局下的市场逻辑

## 命题

作者主张：百年未有之大变局≠全球化死亡。市场本位会沿"避战 / 躲战 / 绕壁垒而行"继续推进。

## 证据

1) 越南 / 墨西哥承接贸易转移；2) ASML / NVIDIA 失去客户；3) 1978 改革开放反向利用价值

## 为何重要

理性经济人仍然在全球流动 — 压抑不到

## 限制与反例

证伪条件：地缘冲突全面升级到能源 / 粮食级别
