# 阅读方式

> 各类阅读法的标准化词汇。配合每本书的读书脑图 `## 阅读方式` 段使用。

## 按线性 vs 非线性

| 方式 | 释义 | 适合 |
|---|---|---|
| **顺序阅读 / Sequential reading** | 从首至尾，不跳读 | 文学、叙事 |
| **检视阅读 / Survey reading** | 看封面、目录、序、摘要、索引 | 第一次拿到一本书 |
| **非线性阅读 / Non-linear reading** | 任意点切入 | 工具书、参考资料 |
| **随机访问 / Random access** | 任意点开始、停止 | 字典、API 文档、维基 |

## 按速度维度

| 方式 | 释义 | 适合 |
|---|---|---|
| **精读 / Close reading** | 一字一句，慢速，注重点与意 | 文学鉴赏；学外语经典 |
| **略读 / Skimming** | 看标题/段首/段尾，抓主旨 | 通识阅读 |
| **扫读 / Scanning** | 找关键词、数字、专有名词 | 资料查找 |
| **速读 / Speed reading** | 拓展视野训练，牺牲理解深度 | 商业类、新闻 |
| **沉浸式阅读 / Immersive reading** | 长时段、连续专注 | 哲学、文学 |
| **批读 / Annotative reading** | 边读边批注 | 学术、研经 |

## 按目的 / 信息密度

| 方式 | 释义 | 适合 |
|---|---|---|
| **主题阅读 / Syntopical reading** | 跨多本书找同一主题的论点 | 写论文、做研究 |
| **比较阅读 / Comparative reading** | 多本书并排对照 | 决策、综述 |
| **批判性阅读 / Critical reading** | 评估作者论证是否成立 | 学术、辩论 |
| **功能性阅读 / Functional reading** | 为解决具体问题 | 手册、说明书 |
| **娱乐性阅读 / Recreational reading** | 为消遣 | 小说、随笔 |

## 中文与古典

- **吟诵 / Chanting & memorization** — 蒙学、古文
- **抄写 / Copying** — 经书临摹（抄经）
- **点读 / Punctuation reading** — 古籍断句
- **集注 / Collected commentaries** — 同一段读多家注
- **评点 / Marginal commentary** — 金圣叹、毛宗岗评点《三国》《水浒》
- **背诵 / Recitation** — 熟读成诵

## 学术 / 宗教传统

- **神圣阅读 / Lectio Divina**（4 步：诵读→默想→祷告→静观）
- **塔木德研究式 / Pilpul** — 经文辩证式精读
- **重复阅读 / Re-reading** — 经典每隔几年再读一次（克尔凯郭尔）
- **布鲁姆斯伯里式 / Bloomsbury close reading** — 一段可以读一下午

## 现代 / 学习法

- **社交阅读 / Social reading** — 多人同步共读，定期讨论
- **听读 / Audio reading** — 用耳朵代替眼睛（通勤 / 视障）
- **碎片阅读 / Fragmented reading** — 利用零碎时间（5–15 分钟）
- **深度阅读 / Deep reading**（对比浅阅读 Shallow reading）— 长时段专注、建构心智模型
- **清单式读书法** — 列已读/在读/想读，做记录
- **FM3 读书法**（成甲《好好学习》） — 读书→笔记→讲给别人听
- **RIA 拆书法** — R 摘抄 / I 释义 / A 应用
- **费曼阅读法 / Feynman technique** — 假装教别人，确认理解
- **康奈尔笔记法 / Cornell** — 阅读 + 5R 笔记
- **卡片笔记法 / Zettelkasten** — 阅读过程生成思想卡

## 数据 / 信息流

- **RSS 推送式 / Feed-based** — 订阅多源聚合读
- **Timeline 阅读** — 按时间排的社交媒体流
- **聚合阅读 / Aggregated reading** — Pocket / Instapaper 类工具收集后集中读
- **新闻懒读 / Lazy news** — 只看摘要

## 与你之前的表达对照

| 你说 | 学术名 |
|---|---|
| 顺序读 | **Sequential / Linear reading**，伴随精读即 **close reading** |
| 跳着读 | **Skimming**（抓主旨） 或 **Skipped reading**（跳过不必要章节） |
| 跳读找特定词 | **Scanning** |
| 看一遍不求细节 | **Survey reading** |
| 反复读 | **Re-reading** / **螺旋阅读** / 哲学里 **Dialectical reading** |
| 先看目录 | **Structural reading**（结构阅读） |

## 推荐组合

- **第一次拿到文学书**：整体阅读 → 检视阅读 → 精读
- **工具书**：索引驱动 + 精读重点 + 扫读其他
- **多本书写论文**：主题阅读 / Syntopical
- **通勤**：听读 + 碎片阅读
- **经典著作反复读**：重复阅读 / Re-reading（间隔 6 个月到 1 年）
