# 领导者决策力

## 命题

决策质量高低 = 衡量管理者是否成功的关键指标。

领导者力 = 高质量决策力 — 不是"做最牛的选择"，而是"持续做可证的过程"。

## 4 个层级

| 层级 | 核心能力 |
|---|---|
| **决策者** | 把握住"决策类型"与"决策场景"，选对决策角色 |
| **决策策略** | 利用反偏见流程 / 助推架构 / 决策工具 |
| **决策影响** | 通过个人决策引导团队 / 组织决策 |
| **决策文化** | 建立组织级纠偏机制与广开言路文化 |

## 应用

- 给团队管理者：决策流程中心 + 红队 / 第三方审计
- 给中层经理：决策风格与骑士精神 → 享受决策的责任感
