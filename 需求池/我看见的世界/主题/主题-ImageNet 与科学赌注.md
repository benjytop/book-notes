# ImageNet 与科学赌注

## 命题

ImageNet = 一个赌 — 足够大 + 足够噪 → 2012 AlexNet 革命实证。

## 证据与案例

1) Caltech 101 失败 → ImageNet 2009 启动（1400 万张图 + 20000 类别）。
2) 2012 AlexNet 用 GPU + ReLU + Dropout → ILSVRC 错误率减半。
3) 同步出现：GPU 算力 + 数据规模 + 算法三件事同时成熟。

## 限制

1) 在其他领域，足够大不一定能引发革命 — GPU + 训练法同步成熟是关键。
2) ImageNet 不能直接迁移到所有领域（如医疗影像 / 时间序列）。
3) 现代大型 Transformer 模型不一定需要"赌足够大"，但确实需要"赌足够大 + 足够长训练"。
