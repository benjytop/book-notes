## 核心论点

1. **关于学习的 4 大误解**——先天智力、失败 = 坏、遗忘 = 失败、多多益善
2. **优化学习的 3 大原则**——专注 / 速度 / 时间（"两步法" + "6 策略"）
3. **学习风格分 3 大类**——视觉 / 听觉 / 动手（VAK 模型）
4. **阅读的 3 个关键技巧**——眼球 / 不回读 / 三阶段读法
5. **笔记的 4 阶段法**——记 → 编辑 → 分析 → 反思
6. **练习 = 进步**——超越"天赋"思维的刻意练习
7. **记忆的"大象方法"**——把抽象信息"感官化"（地点法 / 故事法 / 图解法 / 角色法）
8. **番茄工作法**——25 分钟专注 + 5 分钟休息
9. **即时回馈 vs 间隔重复**——临时抱佛脚的有效原理
10. **学习的目标设定**——SMART 与改进型 SMART
