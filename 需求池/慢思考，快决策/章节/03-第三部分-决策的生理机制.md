# 03 决策的生理机制

## 这一部分主题

大脑与神经系统如何参与决策——为"流程中心"提供生理基础。

## 关键内容

### 双系统决策

- **系统 1 / 斗士 / 侦察兵模式 A**：快速、自动、情绪驱动 — 杏仁核主导
- **系统 2 / 斗士 / 侦察兵模式 B**：慢速、审慎、认知驱动 — 前额叶主导

### 引言：泰勒的中风实验

> 引吉尔·波特·泰勒："与其说人类是"会感受的理性动物"，不如说人类是"会思考的感性动物"。"

> 她的书《My Stroke of Insight》记录自己中风后的身心体验——实证左右脑分工对决策体验的影响。

### 生理学意义

- 长期慢性压力 → 前额叶萎缩 → 系统 2 弱化 → 决策更依赖系统 1
- 急性压力 → 肾上腺素上升 → 反应速度提升但判断精度下降
- 充足睡眠 → 前额叶皮质清理 → 系统 2 重新可用
