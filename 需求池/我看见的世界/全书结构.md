# 全书结构

## 一句话核心命题

**从"我为何学物理"到"AI 该为何学"，这本书是李飞飞把"科学家作为公共知识分子"的身份跋涉写成的自传。**

## 4 部分 + 12 章主题词

| 部分 | 主题词 |
|---|---|
| 第 1 部分 | 逐梦之源（章 1-3） |
| 第 2 部分 | 心智探索（章 4-5） |
| 第 3 部分 | ImageNet 与深度学习突破（章 6-9） |
| 第 4 部分 | 以北极星为镜（章 10-12） |

## 核心论点（10 条）

1. **好奇心与追梦的勇气** — 童年物理梦是李飞飞所有决定的地基
2. **移民的认知优势** — 在新语言 / 新文化中长大的孩子拥有双视角
3. **AI 寒冬下的反周期选择** — 别人不看好 Vision 时，李飞飞坚持
4. **ImageNet 的科学赌注** — 把"足够大"作为直觉假设，让实证去验
5. **GPU + 数据 + 算法三者同步突破** = 深度学习革命
6. **"以人为中心的 AI" 转向** — 母亲病房触发
7. **AI 不可控问题** — 模型发布后社会影响不归研发者"拥有"
8. **AI 科学的下一个北极星** — 用科学 + 人文双轮驱动
9. **女性 / 华裔 / 母亲 的身份** — 多元身份是研究角度多样性的来源
10. **科学家的公共责任** — 学者不能只待在实验室

## 学术依据

- **Geoffrey Hinton**：神经网络 + 反向传播 → 深度学习之父（与李飞飞同代）
- **吴恩达**：与李飞飞同事 + 斯坦福 AI 实验室
- **丹尼尔·卡尼曼**：决策心理（影响"以人为中心"的 AI 设计）
- **维特根斯坦**：语言边界
- **库兹韦尔**：摩尔定律 / 奇点理论

## 全书时间轴（叙事 / 写作）

- 1976：李飞飞生于中国北京
- 1992：12-13 岁与家人移民美国（新泽西州）
- 1995-1999：普林斯顿大学物理学本科
- 1999-2005：加州理工博士（电气工程 + 认知科学）
- 2005-2007：普林斯顿大学助理教授（建立 Princeton Vision Lab）
- 2007-2009：ImageNet 启动（数据众包 + 标注）
- 2009-2012：加入斯坦福大学，创建 Stanford Vision Lab
- 2010：ImageNet 大规模视觉识别挑战赛（ILSVRC）发布
- 2012：AlexNet（Hinton 学生）获胜（GPU 训练 + ReLU + Dropout）
- 2013-2016：深度学习革命全面展开
- 2016：李飞飞宣布从 Google Cloud 卸任 CEO（重回斯坦福）
- 2017-至今：领衔 Stanford HAI（以人为中心的 AI 研究院）
- 2018：清华后保研事件 / 国会议员听证
- 2023：英文版 *The Worlds I See* 出版
- 2024-4：中文版（赵灿译，中信出版集团；据豆瓣 subject/36672955）
