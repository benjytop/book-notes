# 内啡肽

## 定义

> 天然止痛 + 兴奋激素，runner high 来自内啡肽释放。

## 核心命题

> 身体把痛苦 / 强烈刺激转化为愉悦 — 这就是为什么剧烈运动 / 辣食 / 大笑会上瘾。

## 工具 / 证据

8+ 工具：跑 / 笑 / 大笑瑜伽 / 黑巧克力 / 辣食 / 性 / 高强度间歇运动 / 听激烈音乐。

## 限制与反例

但与多巴胺不一样 — 内啡肽更好长期沉淀。
