# 助推（Nudge）

## 命题

> **作者主张**：助推指"用胳膊肘轻触他人，以起到提醒或引起他人注意的作用"。
>
> 它通过改变**选择架构**，不禁止任何选项、不显著改变经济诱因，就能引导人们的行为发生预期变化。

这一概念由 2017 年诺贝尔经济学奖得主 **理查德·H·泰勒** 提出。

## 证据 / 案例

### 超市定价与陈列

- 啤酒、零食放在眼平位置 → 不限购，但增加购买率
- 健身房会员签约 → 自动扣款比主动续费保留率更高

### 主流应用

- 美国"401(k)"自动加入退休金计划
- 英国"养老储蓄自动加入"
- 学校食堂果盘放在更显眼位置

## 关键概念

| 概念 | 含义 |
|---|---|
| **默认选项（Default）** | 决策者倾向于接受默认推荐 |
| **损失规避（Loss Aversion）** | 不想"失去"——助推利用此机制 |
| **简洁信息（Simplification）** | 让选择变得清晰 |
| **反馈（Feedback）** | 告诉选择者"他们的选择会产生什么" |

## 分析

助推 vs 强制 vs 经济激励：

| 方法 | 特点 |
|---|---|
| 强制 | 限制选择（侵犯自由） |
| 激励 | 用钱改变行为（昂贵） |
| 助推 | 改架构，自由仍在（温和） |

**助推动作的边界**——必须是"善意的"、"透明的"。否则会滑向"操纵"。

## 限制与反例

- 部分助推实际上是**不允许说"否"**（如默认器官捐献制度），引发伦理争议
- 助推对**反主流** 人群（reckless / contrarian）效果有限
