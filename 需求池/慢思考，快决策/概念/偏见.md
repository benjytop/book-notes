# 偏见

## 命题

> **作者主张**：偏见是决策的最大天敌。它们来自两个层级：
>
> 1. **个人认知层**（25+ 种认知偏差）
> 2. **组织文化层**（沉默文化、过度自信、责任分散等）

偏见不可"清除"，但可以"识别 + 标记"——这是本书的核心立场。

## 常见偏见类型

| 偏见 | 概念 |
|---|---|
| 锚定效应 | [[锚定效应]] |
| 可得性启发法 | [[可得性启发法]] |
| 框架效应 | [[框架效应]] |
| 确认偏差 | 选择性寻找支持既有观点的信息 |
| 过度自信 | 高估自己判断的准确性 |
| 后见之明 | 事后认为"我早就知道" |
| 损失厌恶 | 等量损失比等量收益更强烈 |

## 证据 / 案例

- **歌德**："偏见缠住了人的性格，就无法克服，因为它们成了人本身的一部分。"
- **大众尾气案**：所有相关工程师都以"先入为主的预期（销量翻三倍）"主导判断——确认偏差的典型。
- **诺曼底气象**：指挥层的气象学家身份压制他们的科学建议——权威 + 沉默文化叠加。

## 分析

本书的"偏见"概念是**卡尼曼/特沃斯基学术谱系**在决策领域的工程化版。它与卡尼曼的双系统模型有一个细微差别：本书更强调**偏见是组织级的产物**，个人偏差部分可被组织流程纠偏。

## 限制与反例

- "流程纠偏"对**慢性隐性**偏见（如内隐偏好）效果弱
- 极端情境下，时间压力让任何纠偏失效
- 偏见有时反而有**保护功能**（如乐观偏见激励创业）
