# 神经元网络

## 定义

> 受到生物神经启发的一类数学模型，含输入 / 加权 / 激活函数 / 输出层

## 核心命题

> 神经网络是 AI 的语言 — 在 Vision / NLP / 跨模态皆然

## 证据与案例

> Hinton 1986 反向传播论文 → 2012 AlexNet（GPU 训练 + ReLU + Dropout）

## 限制与反例

> 近年 Transformer / Mamba / 神经网络即建模 — 神经网络未必是终极范式
