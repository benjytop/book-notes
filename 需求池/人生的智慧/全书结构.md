# 全书结构

## 一句话核心命题

**幸福的根源是人的'自身'，而不是财产或表象 — 90% 取决于人是什么（内在素质），10% 取决于人有什么（外在）。**

## 5 章 + 附录 主题词

| 部分 | 主题词 |
|---|---|
| 第 1 章 | 亚里士多德三分类 (外在之物 / 灵魂 / 身体) |
| 第 2 章 | 人的自身 — 90% 的幸福来源 |
| 第 3 章 | 人所拥有的财产 — 不是越多越好 |
| 第 4 章 | 人所展现的表象 — 不是更重要 |
| 第 5 章 | 建议和格言 — 47 节格言，分 4 部分 |
| 第 6 章 | 人生的各个阶段 |
| 附录 | 叔本华生平轶事 + 哲学影响 |

## 核心论点（10 条）

1. **自身 > 一切** — 90% 的幸福取决于自身
2. **身体健康是其他一切的承载** — 没有健康 = 一切归零
3. **财富够用就好，过多易成负担**
4. **别过分看重别人对你的评价**
5. **学会人群中的部分孤独**
6. **不要对他人抱有期待** — 降低期望 = 降低失望
7. **命运是外在，回应是内在** — 我们能管理的只是后者
8. **不要模仿他人** — 模仿 = 失去自己
9. **保持中道** — 不过分追求亦不过分回避
10. **对命运变动淡然相对** — 漠然不等于冷漠

## 学术依据

- **亚里士多德**：三分类 + 首要律条 (适度)
- **斯多亚学派**：节制 / 淡然 / 命运
- **康德**：批判哲学背景
- **蒙田**：随笔散文 (本世纪欧洲文学之父)
- **伏尔泰 / 歌德 / 荷马**：反复被引用
