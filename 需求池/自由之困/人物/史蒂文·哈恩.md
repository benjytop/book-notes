# 史蒂文·哈恩

**角色 / 身份**：美国历史学家 / 纽约大学教授

## 简介

1979 年在耶鲁大学获得博士。曾任教于特拉华大学 / UCSD / 西北大学 / 宾夕法尼亚大学。研究领域包括美国 19 世纪史 / 奴隶制与废奴运动史 / 资本主义史和南方史。

## 在本书中的贡献

代表作《我们脚下的国家：从奴隶制到大迁徙，南方农村地区的黑人政治斗争》(2003) 获 2004 年度普利策历史奖 + 班克罗夫特奖。
