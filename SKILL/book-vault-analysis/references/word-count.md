# 字数规则 (C12 详细)

## 计算源

**EPUB spine 全量字符数** (不去除扉页/营销/版权/目录)。

```python
spine_chars = sum(len(f.read_text(encoding="utf-8")) for f in spine_dir.glob("*.txt"))
```

## 数字格式

| 范围 | 格式 | 示例 |
|---|---|---|
| ≥ 10,000 | `XX.X 万字` (一位小数) | `21.6 万字` |
| < 10,000 | 整数 + `字` | `8,500 字` |

## 实现

```python
def format_word_count(chars: int) -> str:
    if chars >= 10000:
        # 转为万, 保留一位小数
        value = chars / 10000
        # 整数时不显示小数点
        if value == int(value):
            return f"{int(value)} 万字"
        return f"{value:.1f} 万字"
    else:
        return f"{chars:,} 字"
```

## 位置

MOC「书籍信息」段, 第 9 字段。

```markdown
1. 作者：XXX
2. 译者：XXX
3. 出版社：XXX
...
8. **类型：T1 文学 - 小说**
9. **字数：21.6 万字**     ← 这里
10. 豆瓣评分：X.X / NNNN 人评价
```

## 禁止事项

- ❌ "全文字数" + "正文字数" 双口径
- ❌ "约" / "大约" 模糊词
- ❌ "21.6432 万字" (超过一位小数)
- ❌ 罗马数字 `XXI 万字`
