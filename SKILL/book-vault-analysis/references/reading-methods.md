# 阅读方式 (R1)

每个 vault 必含 `## 阅读方式` 段, 选 4-5 种阅读法。

## 7 种基础阅读法 (最低必选 4 种)

1. **顺序阅读 (Sequential reading)** — 必选
2. **精读 (Close reading)** — 必选
3. **主题阅读 (Syntopical reading)** — 必选
4. **重复阅读 (Re-reading)** — 必选
5. 检视阅读 (Survey reading) — 可选
6. 批判性阅读 (Critical reading) — 可选
7. 比较阅读 (Comparative reading) — 可选

## MOC 嵌入模板

```markdown
## 阅读方式 (7 种阅读法中必选 4-5 种)

1. **顺序阅读 (Sequential reading)** — 叙事性长篇小说, 不能跳过
2. **精读 (Close reading)** — 文学欣赏, 关键段落密集
3. **主题阅读 (Syntopical reading)** — 与《追风筝的人》《群山回唱》对照读
4. **重复阅读 (Re-reading)** — 阿富汗背景, 建议隔 1-2 年重读
5. **检视阅读 (Survey reading)** — 第一次先快速过结构
```
