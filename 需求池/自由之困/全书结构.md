# 全书结构

## 一句话核心命题

**美国从来不是纯粹的自由主义国家 — "非自由主义"是 250 年的幽暗脉络，从殖民地到 BLM 时代贯穿。**

## 核心论点（10 条）

1. 自由主义是 18 世纪发明的叙事武器
2. 等奴制 / 契约奴 / 种族排除 = 制度性安排
3. 内战后种族隔离是联邦政府的妥协与暴力共同制造的
4. 排华法案（1882）是亚裔被压迫的标志案例
5. 工会激进化 → 联邦镇压 → 资本控制
6. 麦卡锡时代 / 反智 → 公开理性的破产
7. 民权运动 = 自由主义的胜利
8. 民权运动后 = 隐性种族主义的延续
9. BLM（2020）= 反思接续
10. 250 年线 = 自由 / 非自由纠缠
