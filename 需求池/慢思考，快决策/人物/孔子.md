# 孔子

> 中国春秋末期思想家（前 551–前 479）。儒家学派创始人。

## 在本书中的角色

**4 步流程古意**。本书引用：

> "子绝四：毋意，毋必，毋固，毋我。"
>   — 《论语·子罕》

## 在本书中的论证作用

- 第 5 部分：作为 4 步流程 "发 / 检 / 展 / 收" 的古典镜像
- "毋意、毋必、毋固、毋我" → 反对"武断 / 教条 / 固执 / 自我中心"——决策中的"空杯思维"
