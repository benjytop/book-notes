# 康德

## 简介

**伊曼努尔·康德** (1724-1804) 是德国哲学家, 古典哲学奠基人. 调和理性主义与经验主义. 三大批判: 纯粹理性 / 实践理性 / 判断力.

## 关键作品

1. 《纯粹理性批判》 (1781)
2. 《实践理性批判》 (1788)
3. 《判断力批判》 (1790)
4. 《永久和平论》

## 核心命题

- 先天综合判断
- 物自体 (Ding an sich) 不可知
- 绝对命令: 道德法则
- 二律背反: 理性的局限
